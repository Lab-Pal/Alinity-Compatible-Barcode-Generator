Quick Start Guide

1

Select Container Type

Choose the container that matches your QC workflow

2

Enter QC Information

Required fields:

  • Lot Number
  • At least one QC level

    • Important: Enter the Lot Number exactly as registered in your Alinity.

    To Verify the lot number:
    Create Order → Control → Select Control Name → Copy the Control Lot

    3

    Choose Output Mode

    Avery 5195 (60 labels)

  • Labels are automatically divided among selected QC levels.
  • Example: 2 levels selected = 30 labels each on one sheet

    • Custom (inch)

  • Designed for thermal label printers.
  • Width and height adjust label positioning only - they do not change the barcode size
  • The chosen quantity is produced for each QC level.

    • Show Borders around each label to help with alignment or cutting.

    4

    Generate & Print

  • Click Generate Barcodes.
  • Click Download PDF
  • Print PDF using Actual Size / 100% Scale.

    • ×

    🎉 Barcode Labels Generated

    Your PDF is ready to download.

    Print using Actual Size / 100% Scale for proper Avery 5195 alignment.
